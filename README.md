## 简介
> **Note**  
> 一切开发旨在学习，请勿用于非法用途

基于 Nonebot2 和 go-cqhttp 开发，以 sqlite3 作为数据库的娱乐机器人

## 特色
* 使用 NoneBot2 进行项目底层构建.
* 使用 go-cqhttp 作为默认协议端.
* 互通 QQ、QQ频道 等平台
* 独立ID，更方便管理与互通数据
* 全新的权限系统，不用重启便可自定义 Bot管理员 和 Bot协助者
* Coming soon...
